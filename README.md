# Data-Visualization-and-Reporting

## Overview
This project involves transforming and analyzing sales and purchased data of UHospital by using Power BI Desktop. 
The messy and unclear format of data, requiring thorough data cleaning, transformation, and visualization to build a dashboard for business insights.

## Dashboard 
These are the following informations included:
1. Sales Summary Dashboard
   - Total Sales by period
   - Sales by region, channel, or sales team
   - Top selling products or services
     
2. Sales Trend Analysis
   - Year-over-year (YOY) and month-over-month (MOM) comparisons
   - Seasonality Pattern
   - Sales Target vs Actuals
   - Performance vs Forcasted Goals
     
3. Sales Performance by Type/Class
   - OEM:
     - Local OEM
     - Export OEM
   - OBM:
     - Local OBM
     - Export OBM
     - Online Sales

4. Drill-down by product, region, or salesperson

## Tools and Technologies
- Power BI Desktop
- Dax
- Power Query
